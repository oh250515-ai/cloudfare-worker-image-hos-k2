Text & chat

GLM 4.7 Flash

@cf/zai-org/glm-4.7-flash

Mạnh: nhanh, đa ngôn ngữ, context 131k, reasoning và tools.

Yếu: không có vision.

Dùng: chat tiếng Việt, tóm tắt, phân loại.

GPT OSS 20B

@cf/openai/gpt-oss-20b

Mạnh: reasoning, agentic, 128k.

Yếu: input riêng, gọi qua /v1/run.

Dùng: analysis và tool workflows.

Code

GLM 5.2

@cf/zai-org/glm-5.2

Mạnh: flagship coding, 262k, tools.

Yếu: đắt và chậm.

Dùng: refactor lớn, coding agents.

Kimi K2.7 Code

@cf/moonshotai/kimi-k2.7-code

Mạnh: code + vision + tools, 262k.

Yếu: đắt cho snippet nhỏ.

Dùng: repo lớn, code kèm screenshot.

Vision & OCR

Mistral Small 3.1

@cf/mistralai/mistral-small-3.1-24b-instruct

Mạnh: vision + text, 128k, structured extraction.

Yếu: chậm hơn model nhỏ.

Dùng: screenshot WinForms dày chữ.

Gemma 4 26B

@cf/google/gemma-4-26b-a4b-it

Mạnh: vision, reasoning, 256k, giá thấp.

Yếu: cần benchmark OCR tiếng Việt.

Dùng: multimodal cân bằng.

Llama 4 Scout

@cf/meta/llama-4-scout-17b-16e-instruct

Mạnh: multimodal native, 131k, tools.

Yếu: không chuyên OCR.

Dùng: image Q&A, UI reasoning.

Moondream 3.1

@cf/moondream/moondream3.1-9B-A2B

Mạnh: nhỏ, nhanh, point/detect.

Yếu: 2/6 anchors trên ảnh DHG dày chữ, có loop.

Dùng: ảnh thưa chữ và caption nhanh.

Ví dụ thật

# Text
curl $BASE/v1/text -H 'content-type: application/json' \
  -d '{"model":"@cf/zai-org/glm-4.7-flash","prompt":"Tóm tắt..."}'

# Code
curl $BASE/v1/code -H 'content-type: application/json' \
  -d '{"model":"@cf/zai-org/glm-5.2","prompt":"Review đoạn code..."}'

# Model có input riêng
curl $BASE/v1/run -H 'content-type: application/json' \
  -d '{"model":"@cf/openai/gpt-oss-20b","input":{"instructions":"Trả lời ngắn","input":"CAP theorem là gì?"}}'
Policy: allowedModels:"*" nhận mọi ID hợp lệ dạng @cf/author/model. Production nên dùng glob hẹp hoặc danh sách cụ thể.
Nguồn chuẩn: Cloudflare model catalog và changelog. Kiểm tra deprecation trước mỗi release.
