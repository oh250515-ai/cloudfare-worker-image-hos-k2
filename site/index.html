Cloudflare Worker + Workers AI

Ảnh, text, code. Chung một API.

Worker mỏng gọi mọi model trên Workers AI: trích JSON từ ảnh, sinh text/code/chat, và benchmark nhiều model ngay trong một request. Cấu hình qua một secret duy nhất.

Mở PlaygroundDanh mục modelGitHub

Endpoints

MethodPathCông dụng
GET/healthKiểm tra sống
GET/v1/modelsModel mặc định, allowlist, adapters
POST/v1/extractẢnh (URL/base64) → JSON + rawText + annotations
POST/v1/textSinh văn bản từ prompt hoặc messages
POST/v1/codeSinh code (model chuyên code)
POST/v1/chatChat theo messages
POST/v1/runTruyền thẳng input cho model bất kỳ

Ví dụ

curl -s $BASE/v1/text -H 'content-type: application/json' \
  -d '{"prompt":"Viết một câu chào","parameters":{"max_tokens":64}}'

# So sánh 2 model trong một lần gọi
curl -s $BASE/v1/text -H 'content-type: application/json' \
  -d '{"prompt":"Say hello","benchmark":{"models":["@cf/meta/llama-3.1-8b-instruct","@cf/zai-org/glm-4.7-flash"],"runs":2}}'
